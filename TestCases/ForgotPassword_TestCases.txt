Test Case ID: TC_FP_001
Title: Forgot password with registered email
Precondition: Email exists in system
Steps:
1. Open login page
2. Click Forgot Password
3. Enter registered email
4. Submit
Expected Result: Reset link/OTP sent successfully

Test Case ID: TC_FP_002
Title: Forgot password with unregistered email
Precondition: Email not in system
Steps:
1. Open Forgot Password
2. Enter unregistered email
3. Submit
Expected Result: Proper error or safe message shown

Test Case ID: TC_FP_003
Title: Reset password with mismatched confirm password
Precondition: User has valid reset link/OTP
Steps:
1. Open reset password page
2. Enter new password
3. Enter different confirm password
4. Submit
Expected Result: Validation error, password not changed
